## معرفی

این قالب یک فایل HTML مستقل (`index.html`) است که جایگزین صفحه اشتراک پیش‌فرض پاسارگارد می‌شود.  
تمام HTML، CSS و JavaScript داخل همان فایل قرار دارد؛ **نیازی به npm، webpack یا build نیست.**

صفحه شامل سه بخش اصلی است:

| بخش | توضیح |
|-----|--------|
| **اطلاعات حساب** | وضعیت اشتراک، حجم، زمان باقی‌مانده، نمودار مصرف |
| **کانفیگ‌ها** | لینک اشتراک، کپی، QR و دانلود WireGuard |
| **اپلیکیشن‌ها** | لیست برنامه‌ها بر اساس سیستم‌عامل |

---

## ویژگی‌ها

- طراحی **شیشه‌ای (Glassmorphism)**، مدرن و بهینه برای موبایل
- پشتیبانی از **فارسی (پیش‌فرض) و انگلیسی** با سوئیچ زبان
- تم **تاریک و روشن** با ذخیره انتخاب کاربر
- نوار پایین موبایل با **رنگ‌بندی اختصاصی** هر تب در حالت فعال
- نمایش اطلاعات اشتراک: حجم مصرفی، باقی‌مانده، تاریخ انقضا، آخرین اتصال، IP، ظرفیت دستگاه
- **نمودار مصرف** روزانه و هفتگی
- **هشدار** کم بودن حجم یا نزدیک شدن به انقضا
- دریافت خودکار **اعلان و اپلیکیشن‌ها** از تنظیمات پنل
- کپی تکی / کپی همه کانفیگ‌ها + **QR Code**
- دانلود کانفیگ **WireGuard**
- تشخیص سیستم‌عامل و اولویت‌دهی به OS کاربر
- فونت **وزیرمتن** از CDN
- بهینه‌سازی عملکرد برای دستگاه‌های ضعیف (`data-perf="lite"`)

## ⚠️ شخصی‌سازی برند — اجباری

> **قبل از استفاده در محیط واقعی، حتماً برند پیش‌فرض را تغییر دهید.**  
> مقادیر فعلی (`MrClock Template` / `پنل اشتراک`) فقط برای نمایش دمو هستند و **نباید** به کاربر نهایی نشان داده شوند.

### کجا را ویرایش کنم؟

فایل `index.html` را باز کنید و با جستجو (`Ctrl+F`) عبارت **`DEFAULT_BRAND`** را پیدا کنید.  
بلوک تنظیمات برند در **حدود خط ۱۹۶۴** قرار دارد:

```javascript
var DEFAULT_BRAND = {
  name: "MrClock Template",       // ← نام برند
  subtitle: {
    fa: "پنل اشتراک",             // ← شعار فارسی
    en: "Subscription panel"      // ← شعار انگلیسی
  },
  logoUrl: ""                     // ← URL لوگو (خالی = آیکن پیش‌فرض)
};
```

---

### ۱. نام برند (`name`) — خط ۱۹۶۵

| | |
|---|---|
| **فیلد** | `name` |
| **نمایش** | عنوان بزرگ بالای صفحه (کنار لوگو) |
| **مثال** | `"فروشگاه VPN من"` |

```javascript
name: "فروشگاه VPN من",
```

---

### ۲. شعار (`subtitle`) — خطوط ۱۹۶۶ تا ۱۹۶۹

| | |
|---|---|
| **فیلد** | `subtitle.fa` و `subtitle.en` |
| **نمایش** | متن کوچک زیر نام برند |
| **رفتار** | با تغییر زبان، شعار همان زبان نمایش داده می‌شود |

```javascript
subtitle: {
  fa: "پنل مدیریت اشتراک",
  en: "Subscription management"
},
```

> اگر فقط یک زبان دارید، هر دو مقدار را یکسان بنویسید.

---

### ۳. لوگو (`logoUrl`) — خط ۱۹۷۰

| | |
|---|---|
| **فیلد** | `logoUrl` |
| **نمایش** | تصویر داخل دایره کنار نام برند |
| **خالی** | `" "` → آیکن شبکه پیش‌فرض نمایش داده می‌شود |
| **پر** | آدرس کامل تصویر → لوگوی شما جایگزین می‌شود |

```javascript
logoUrl: "https://example.com/logo.png"
```

#### نکات مهم لوگو

| نکته | توضیح |
|------|--------|
| **فرمت** | PNG، SVG، WebP یا JPG |
| **ابعاد پیشنهادی** | مربع، حداقل `128×128` پیکسل |
| **دسترسی** | URL باید از مرورگر کاربر **قابل دسترسی** باشد (HTTPS توصیه می‌شود) |
| **CORS** | اگر لوگو روی دامنه دیگر است، سرور باید اجازه بارگذاری تصویر بدهد |
| **هاست محلی** | می‌توانید لوگو را روی همان دامنه پنل آپلود کنید: `"/static/logo.png"` |

#### مثال‌های رایج

```javascript
// لوگو روی همان سرور پنل
logoUrl: "/static/my-logo.png"

// لوگو از CDN یا هاست خارجی
logoUrl: "https://cdn.example.com/brand/logo.svg"
```

---

### ۴. عنوان تب مرورگر (اختیاری) — خط ۷

```html
<title>پنل کاربری</title>
```

این متن روی **تب مرورگر** نمایش داده می‌شود. توصیه می‌شود با نام برند یکسان باشد:

```html
<title>فروشگاه VPN من</title>
```

---

### ۵. روش جایگزین — Override بدون ویرایش مستقیم

اگر نمی‌خواهید داخل `index.html` تغییر دهید، **قبل از** اجرای اسکریپت اصلی، یکی از این متغیرها را در `<head>` تنظیم کنید:

```html
<script>
  window.MRCLOCK_BRAND = {
    name: "برند من",
    subtitle: { fa: "شعار فارسی", en: "English tagline" },
    logoUrl: "https://example.com/logo.png"
  };
</script>
```

اولویت خواندن برند (خط ۱۹۷۳):

```
window.MRCLOCK_BRAND  →  window.PASARGUARD_BRAND  →  window.PANEL_BRAND  →  DEFAULT_BRAND
```

---

### ✅ چک‌لیست قبل از انتشار

- [ ] `name` به نام فروشگاه / سرویس شما تغییر کرده
- [ ] `subtitle.fa` و `subtitle.en` شخصی‌سازی شده
- [ ] `logoUrl` به لوگوی واقعی شما اشاره می‌کند (یا عمداً خالی مانده)
- [ ] `<title>` در خط ۷ به‌روز شده
- [ ] صفحه در مرورگر موبایل و دسکتاپ تست شده
- [ ] لوگو بدون خطا بارگذاری می‌شود

## نصب روی سرور پاسارگارد

### مرحله ۱ — آپلود قالب

وارد سرور شوید و فایل `index.html` را در مسیر قالب اشتراک قرار دهید:

```bash
sudo mkdir -p /var/lib/pasarguard/templates/subscription/
sudo cp index.html /var/lib/pasarguard/templates/subscription/index.html
```

یا با `wget` از مخزن GitHub (در صورت انتشار):

```bash
sudo mkdir -p /var/lib/pasarguard/templates/subscription/
sudo wget -N -O /var/lib/pasarguard/templates/subscription/index.html \
  https://raw.githubusercontent.com/YOUR_USER/YOUR_REPO/main/index.html
```

> **یادآوری:** قبل از آپلود، [شخصی‌سازی برند](#️-شخصی‌سازی-برند--اجباری) را انجام دهید.

---

### مرحله ۲ — تنظیم `.env`

```bash
sudo nano /opt/pasarguard/.env
```

مقادیر زیر را اضافه یا اصلاح کنید:

```env
CUSTOM_TEMPLATES_DIRECTORY="/var/lib/pasarguard/templates/"
SUBSCRIPTION_PAGE_TEMPLATE="subscription/index.html"
```

---

### مرحله ۳ — ریستارت

```bash
sudo pasarguard restart
```

---

### مرحله ۴ — تست

1. لینک اشتراک یک کاربر تست را در مرورگر باز کنید.
2. نام برند، شعار و لوگوی خود را در هدر بررسی کنید.
3. سه تب (حساب / کانفیگ / اپ) و تم روشن/تاریک را تست کنید.

---

## نصب روی هاست

اگر نسخه PHP/هاست جداگانه دارید:

1. فایل‌های پروژه را در **root هاست** آپلود کنید.
2. آدرس پنل را در فایل تنظیمات (مثلاً `index.php`) مطابق مستندات همان نسخه تنظیم کنید:

```javascript
const BASE_URL = 'https://panel.example.com:PORT';
```

3. **برند را در `index.html`** مطابق بخش [شخصی‌سازی برند](#️-شخصی‌سازی-برند--اجباری) تغییر دهید.

> فایل `index.html` این مخزن نسخه مستقیم پاسارگارد است. برای هاست، از بسته Host Version پروژه استفاده کنید (در صورت وجود).

## تنظیم اعلان و اپلیکیشن‌ها از پنل

این موارد **از داخل فایل قالب ویرایش نمی‌شوند** — پنل پاسارگارد منبع داده است.

1. وارد **پنل پاسارگارد** شوید.
2. به **تنظیمات → اشتراک** بروید.
3. **متن اعلان** و در صورت نیاز **لینک پشتیبانی** را تنظیم کنید.
4. در بخش **اپلیکیشن‌ها** برنامه‌ها را اضافه یا ویرایش کنید.

قالب به‌صورت خودکار:

- اپ‌ها را بر اساس **سیستم‌عامل** گروه‌بندی می‌کند
- OS تشخیص‌داده‌شده کاربر را **اولویت** می‌دهد
- آیکن، لینک دانلود و دکمه **افزودن اشتراک** را از پنل می‌خواند

## سؤالات متداول

<details>
<summary><strong>لوگو نمایش داده نمی‌شود</strong></summary>

- URL را مستقیم در مرورگر باز کنید — باید تصویر دیده شود.
- از HTTPS استفاده کنید.
- اگر فایل روی دامنه دیگر است، CORS را بررسی کنید.
- فرمت PNG یا SVG معمولاً کم‌دردسرتر است.

</details>

<details>
<summary><strong>هنوز «MrClock Template» می‌بینم</strong></summary>

- مطمئن شوید `DEFAULT_BRAND` را ویرایش کرده‌اید.
- کش مرورگر را پاک کنید (`Ctrl+Shift+R`).
- اگر `window.MRCLOCK_BRAND` جایی تعریف شده، اولویت با آن است.

</details>

<details>
<summary><strong>آیا می‌توانم فقط فارسی داشته باشم؟</strong></summary>

بله. زبان پیش‌فرض فارسی است. دکمه زبان در هدر برای سوئیچ به انگلیسی است؛ حذف آن نیاز به ویرایش HTML دارد.

</details>

<details>
<summary><strong>فایل چند خط است؟ برند کجاست؟</strong></summary>

فایل تک‌فایلی و حدود ۳۳۰۰ خط. همیشه با جستجوی <code>DEFAULT_BRAND</code> به تنظیمات برند برسید — وابسته به شماره خط نیست.

</details>
